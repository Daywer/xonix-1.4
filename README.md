# Xonix 1.4

A classic **Xonix** arcade game clone.

Классическая аркадная игра **Xonix**.

---

## 🇬🇧 English

### Build

To compile the game, run:

```bash
make -s Makefile.std
```

### Run

Start the game with:

```bash
./xonix <level_number>
```

Where:

- `<level_number>` — a number from `1` to `9`

Example:

```bash
./xonix 1
```

```bash
./xonix 5
```

## Gameplay

Move around the field, capture territory, and avoid enemies.

Перемещайтесь по полю, захватывайте территорию и избегайте врагов.
